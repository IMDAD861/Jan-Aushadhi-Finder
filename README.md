# Jan Aushadhi Finder New

Jan Aushadhi Finder is an Android healthcare application built using Kotlin and Jetpack Compose. The app helps users find affordable generic medicines, compare branded vs generic prices, calculate savings, scan prescriptions using OCR (GenAI), and locate nearby Jan Aushadhi stores.

## Features

* OCR-based Prescription Scanner (GenAI)
* 500+ Medicine Database
* Brand to Generic Medicine Search
* Medicine Price Comparison
* Savings Calculator
* Nearby Store Locator
* Refill Reminder Tracker
* Kannada & English Language Support
* Modern Jetpack Compose UI

## Tech Stack

* Kotlin
* Jetpack Compose
* Material 3
* Google ML Kit OCR
* Gson JSON Parsing

## Project Objective

The app aims to improve healthcare affordability by helping users discover low-cost generic medicines available through Jan Aushadhi Kendras.
